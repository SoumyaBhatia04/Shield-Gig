# Shield-Gig: AI-Powered Micro Insurance

## Problem
Delivery workers lose income during rain, pollution, or curfews.

## Solution
An AI-powered insurance system that provides automatic payouts when disruptions happen.

## Target Users
Zomato / Swiggy delivery partners

## Workflow
1. Worker registers
2. AI calculates weekly premium
3. System tracks weather
4. Disruption detected
5. Claim auto triggered
6. Payment sent

## Weekly Premium Model
- Low risk: ₹20/week
- Medium risk: ₹30/week
- High risk: ₹40/week

## Parametric Triggers
- Rain > 50mm
- AQI > 300
- Extreme heat
- City lockdown

## AI Usage
- Risk prediction
- Dynamic pricing
- Fraud detection

## Tech Stack
Frontend: React  
Backend: Node.js  
Database: MongoDB  
AI: Python  

## Prototype
Basic UI screens showing registration, premium calculation, and claim system.
